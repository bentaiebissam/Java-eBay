This contains a Java web application coded using the Java Spring MVC

The website is a betting interface for products. 
Users are allowed to:
-Create an account
-Sell products
-Buy products
-Change their balance
-View the list of products
-Bet on a product
-View the history of actions made on their account.

The website utilies:
-Java JSTL
-Managed Beans
-Converters and Validators
-Session
-Cookies

The database used is the MySQL "eBay.sql"
